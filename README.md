## Setup
Open the "inc/config.php" file and fill in the Host Name, Username, Password, and Database Name.

## Endpoints
```
annotations  - GET/POST
    /groups  - GET/POST/PUT
    /byId    - GET/PUT
    /byGroup - GET
    /byUrl   - GET
```
